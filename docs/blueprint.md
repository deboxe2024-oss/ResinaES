# **App Name**: Resina Master

## Core Features:

- Content Cloning: Clone the exact content from the provided URL (https://resinaemcasa.netlify.app/), including text, images, and layout structures. Use an AI tool to adjust layout for multiple screen sizes.
- Layout Preservation: Maintain the original site's layout and visual hierarchy to ensure accurate replication.
- Asset Management: Store the extracted assets from the site.
- Responsive Adaptation: Automatically adapt the cloned content for different screen sizes and devices, ensuring a responsive design.

## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F), inspired by the rich hues of resin art. 
- Background color: Soft beige (#F8F9FA), a desaturated version of the primary, to provide a gentle backdrop and highlight the content.
- Accent color: Muted gold (#D4AC0D) to complement the terracotta and add a touch of sophistication.
- Headline font: 'Belleza' sans-serif font for headings to evoke artistry and design.
- Body font: 'Alegreya' serif font for body text for a pleasant reading experience.
- Use hand-drawn style icons related to art and crafts.
- Grid-based layout to maintain order and balance, mirroring the precision in resin art. Ensure elements are evenly spaced.
